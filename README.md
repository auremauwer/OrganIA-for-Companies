## 🚀 Cómo activar GitHub Pages

### Opción 1: Usar este repositorio

1. **Crea un nuevo repositorio en GitHub** llamado `tu-usuario.github.io`

2. **Clone y configure**:
   ```bash
   git clone https://github.com/tu-usuario/tu-usuario.github.io.git
   cd tu-usuario.github.io
   ```

3. **Copia los archivos**:
   - Copia `index.html` al repositorio
   - Copia este `README.md`

4. **Haz commit y push**:
   ```bash
   git add .
   git commit -m "Agregar organigrama"
   git push origin main
   ```

5. **Accede** a: `https://tu-usuario.github.io`

### Opción 2: Usar en un repositorio existente

1. **En un repositorio cualquiera**, ve a Settings → Pages

2. **Selecciona**:
   - Source: Deploy from a branch
   - Branch: main (o tu rama)
   - Folder: / (root)

3. **Guarda** y espera 1-2 minutos

4. **Tu página estará** en: `https://tu-usuario.github.io/nombre-repo`

## 🎨 Personalización

Puedes editar el archivo `index.html`:

- **Cambiar nombres y roles**: Busca las etiquetas `<div class="name">` y `<div class="role">`
- **Cambiar colores**: Modifica los `background: linear-gradient(...)` en la sección `<style>`
- **Cambiar estructura**: Añade o elimina `<div class="position">` según necesites
